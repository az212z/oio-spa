# OiO Nail Spa — موقع + نظام حجوزات وإدارة

نظام حجوزات متكامل لصالون أظافر، يعمل على استضافة ثابتة (GitHub Pages) بدون خادم،
مع قاعدة بيانات سحابية مجانية (Firebase) للمزامنة الفورية بين العميلات والإدارة.

## المكوّنات
| الملف | الوصف |
|------|-------|
| `index.html` | الموقع التعريفي + نظام حجز من ٤ خطوات |
| `admin.html` | لوحة الإدارة (تقويم + قائمة) — قابلة للتثبيت كتطبيق |
| `firebase-config.js` | الإعدادات + كتالوج الخدمات + محرك الجدولة وطبقة البيانات |
| `manifest.json`, `icon.svg` | بيانات تطبيق الإدارة (PWA) |
| `SETUP.md` | دليل ربط Firebase خطوة بخطوة |

## المزايا
**للعميلة (الموقع):**
- حجز من الجوال: اختيار الخدمات ← اليوم والوقت ← البيانات ← تأكيد فوري برقم حجز.
- الأوقات المتاحة تُحسب آليًا حسب مدة الخدمة + الحجوزات + الحظر + عدد الفنيات.
- واتساب متاح كخيار إضافي.

**للإدارة (التشغيل اليومي):**
- 📅 **تقويم** بأعمدة الفنيات × الوقت، بالألوان وخط الوقت الحالي والتنقّل بين الأيام.
- 📋 **قائمة** بكل الحجوزات لحظيًا، فلترة وبحث وتصدير CSV.
- **حجز يدوي** للعميلات الحاضرات/الهاتفيات، و**حظر أوقات** (استراحة/تنظيف) ينعكس فورًا على الموقع.
- تفاصيل كل حجز: تغيير الحالة، تغيير الفنية، اتصال/واتساب، حذف.
- إحصائيات: حجوزات اليوم/الأسبوع، قيد الانتظار، إيراد الشهر.

## التشغيل
1. **تجربة فورية:** افتحي الموقع مباشرة — يعمل محليًا على الجهاز (بيانات الجهاز فقط).
2. **التشغيل الحقيقي (مزامنة بين الأجهزة):** اتبعي `SETUP.md` لربط Firebase (≈ ٥ دقائق).
   هذه خطوة إلزامية لكي يصل حجز العميلة من جوالها إلى تقويم الصالون.

## التخصيص (في `firebase-config.js`)
أسماء الفنيات وألوانها، ساعات الدوام، مدة المواعيد، أيام الإغلاق، أرقام التواصل،
وكتالوج الخدمات والأسعار — كلها قابلة للتعديل من مكان واحد.

## تثبيت لوحة الإدارة كتطبيق
من متصفح التابلت/الجوال: افتحي `admin.html` ← قائمة المتصفح ← «إضافة إلى الشاشة الرئيسية».
